# Netflix Clone (Frontend)

This project is a Netflix-style landing page UI clone made using HTML, CSS, and Vanilla JavaScript. It recreates the look and layout of the Netflix India homepage with a hero banner, trending section, feature highlights, FAQ accordion, and a footer.  
It is built only for learning, practice, and portfolio use.

Disclaimer: This is not the official Netflix website and does not provide streaming. Netflix is a trademark of Netflix, Inc.

## What this project is
A static frontend clone (UI) that focuses on:
- Clean Netflix-like design
- Section-based layout similar to the original landing page
- Simple JavaScript interaction for the FAQ section

## What you will see in the website
- Hero section with background image, title text, and email input CTA button >
- Trending posters section (image-based row layout)
- Feature cards section: “More reasons to join”
- FAQ section with expand/collapse questions
- Footer with useful links and language selector

## Features
- Hero banner with overlay and call-to-action form
- Trending section with posters (image grid/row)
- Feature highlight cards with icons
- FAQ accordion interaction (open/close)
- Simple, structured footer


## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run Locally
### Option 1: Open directly
1. Download or clone this repository
2. Open `index.html` in your browser

### Option 2: VS Code Live Server (recommended)
1. Open the folder in VS Code
2. Install the "Live Server" extension
3. Right click on `index.html` → "Open with Live Server"


## How the FAQ Accordion Works
The FAQ section uses JavaScript to toggle answers:
- When you click a question, the answer container expands
- Clicking again collapses it

This logic is written in `script.js`.
